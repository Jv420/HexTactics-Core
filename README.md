# HexTactics Core

HexTactics Core is de centrale basis voor alle HexTactics-projecten.

## Ecosysteem

HexTactics wordt het hoofdmerk. De bestaande projecten blijven hun eigen naam houden, maar worden uitgebracht als onderdeel van HexTactics.

```text
HexTactics
├── Gaming
│   ├── DynathiSMP
│   ├── DelfzijlRP
│   ├── DelfzijlCity
│   └── AppingedamSCO
├── Development
│   ├── Websites
│   ├── Webshops
│   ├── Discord-bots
│   └── Servertools
└── Infrastructure
    ├── Monitoring
    ├── Back-ups
    ├── Compute nodes
    └── Deployment tools
```

## Doel van deze repository

Deze repository bevat de centrale architectuur, gedeelde code, documentatie en configuratievoorbeelden voor het volledige HexTactics-ecosysteem.

## Belangrijke regels

- Geen wachtwoorden, tokens, private keys of wallet seeds committen.
- Alleen `.env.example` bestanden in GitHub plaatsen.
- Projectspecifieke code blijft in een eigen app- of service-map.
- HexTactics Core blijft de centrale bron voor gedeelde componenten.
